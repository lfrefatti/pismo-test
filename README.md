# Credit Card Payments Processing

## Requirements

  * JDK 8
  * Maven 3.5.0
  * This services uses in memory H2 database

## Running this service

```bash
  >> mvn clean package
  >> java -jar /target/payments-0.0.1-SNAPSHOT.jar
```

## Running tests

```bash
  >> mvn clean verify
```
